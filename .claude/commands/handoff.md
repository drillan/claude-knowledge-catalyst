# プロジェクト引き継ぎコマンド

@.claude/handoff/CLAUDE.md に従って、次のファイルを生成してください

```
.claude/handoff/current/
├── progress.md          # ✅ 完了事項・実装進捗
├── issues.md           # ⚠️ 課題・問題・技術的困難
├── files.md            # 📁 変更ファイル記録（最重要）
└── review-requests.md  # 🤔 Claude相談内容
```
